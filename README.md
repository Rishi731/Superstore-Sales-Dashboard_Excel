# Superstore-Sales-Dashboard_Excel
Developed an Excel dashboard using 9,900+ Superstore sales records to analyze total sales, quantities, order trends, and customer segments across regions, categories, and years. The data was fully cleaned, transformed, and modeled using Power Pivot for accurate and efficient analysis.

Highlights
- Tracks total sales, sales average, transaction count, and total quantity ordered.
- Breaks down ship-mode performance across First Class, Second Class, Same Day, and Standard Class.
- Shows region-wise transactions for Central, East, South, and West.
- Displays monthly quantity trends from January to December.
- Highlights top 5 cities by total sales.
- Provides category-wise transactions for Technology, Office Supplies, and Furniture.
- Shows quantity distribution across Consumer, Corporate, and Home Office segments.
- Supports fast filtering with region and year slicers.
